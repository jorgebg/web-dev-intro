Cascade Style Sheets (CSS)

CSS 1: Introduction

Introduction

CSS (Cascading Style Sheets) is the code that styles web content. It is what you use to selectively style HTML elements. For example, the following code targets the p tags (paragraph text), setting their font color to white and the background color to black:

p {
  color: white;
  background-color: black;
}

This will change the color of the background and font of all the paragraphs in the document (all the p elements).

The CSS code must be inside a <style> tag in the HTML document:

<html>
  <head>
    <style>
      p {
        color: white;
        background-color: black;
      }
    </style>
  </head>
...

It's recommended to place the style tag inside the head tag, so the styles are loaded before the body is rendered.

  5. Upload the HTML file.

CSS 2: Styling text

Introduction

In the previous activity, we have styled all elements based on their HTML tag names (p and body).

To select a subset of the elements without changing the others, you can add a class attribute to the tags and then target that class in your CSS. Elements with the class attribute can be targeted in CSS using a selector following the format .class { ... }. For example:

<style>
  .special {
    font-weight: bold;
  }
</style>
...
<ul>
  <li>Item one</li>
  <li class="special">Item two</li>
</ul>

This will set the style of the second item of the list as bold:

  • Item one
  • Item two

CSS 3: The box model

Introduction

All HTML elements can be considered boxes. Each box has five modifiable CSS properties:

  • height and width describes the dimensions of the content.
  • padding describes the space between the content and the border of the box (makes the element bigger or smaller).
  • border is any kind of line surrounding the box.
  • margin is the space around the border (separation with other elements).

For example:

<style>
  p {
    padding: 5px;
    margin: 5px;
    border: 1px solid green;
  }
</style>

This will give a padding, margin and border to all the paragraphs of the document. You can see the result here.

JavaScript (JS)

JS 1: Introduction

Introduction

JavaScript (JS) is a coding language used to make webpages interactive.

The code must be inside a <script> tag in the HTML document:

<html>
  <head>
    ...
  </head>
  <body>
  ...
    <script>
      JS CODE GOES HERE
    </script>
  </body>
</html>

JavaScript has many functions that allows us to manipulate the elements of the HTML document (along with other things).

For example, the following code calls the function document.body.append, which adds a given text at the end of the body element of the HTML document:

document.body.append("Welcome")

When applied to the HTML Template, the text Welcome will be placed under the initial paragraph:

🏛️ Hello, student!

Welcome

All the functions in JavaScript are called with the following syntax: name(parameter). The code can make multiple function calls, in multiple lines. Each line is executed one after the other, in the same order as they are written.

For example, if we add another call after the previous code:

document.body.append("Welcome")
document.body.append(" to JS.")

The document will end up like this:

🏛️ Hello, student!

Welcome to JS.

It is recommended to place the script tag at the end of the body tag, so the script is executed after all the HTML elements are shown.

JS 2: Variables

Introduction

Variables are names for values (such as numbers or text). We use them in JS for storing data. They are defined like this:

let fruit = "Pear"
  • let is the word that indicates we are defining a variable.
  • fruit is an arbitrary name we are giving to the variable.
  • = indicates we are going to give a value to the fruit variable.
  • "Pear" is the value of the variable.

We can use variables as parameters in functions. For example, we could call the function we saw in the previous practical like this:

let greeting = "Welcome"
document.body.append(greeting)

This would yield the same result we got in the previous practical:

🏛️ Hello, student!

Welcome

Functions can also return values that we can store in variables. For example, the Date function returns the current date and time. The following code stores the current date and time inside a variable named today.

let today = Date()

JS 3: Objects

Introduction

One special thing about variables we saw in the previous practical is that they can contain any kind of data, not just text or numbers. Variables can also contain functions and objects.

Objects are collections of several variables and functions, and they can represent the elements of the HTML document. For example, document.body.append (the function we used in the previous practicals for appending text to the body) is part of the document object.

The document object is the root object that represents the whole HTML document.

Along with many others, the document object has a function for finding HTML elements of the document: document.querySelector(selector). It takes a CSS selector as a text parameter and returns an object representing the HTML element selected.

For example, if we run the following code in our HTML template:

let element = document.querySelector("p")

Then an object representing the paragraph of the document would be stored inside the element variable. (Remember the template has a single p paragraph containing the text "🏛️ Hello, student!").

The, we can manipulate the text of the element changing its propertu textContent:

let element = document.querySelector("p")
element.textContent = "🏞️ It's adventure time!"

This code would transform the HTML from this:

🏛️ Hello, student!

To this:

🏞️ It's adventure time!

JS 4: Functions and events

Introduction

In JS we can define our own functions following this syntax:

function welcome() {
  let text = "Welcome"
  document.body.append(text)
}
  • function is the word that indicates we are defining a function.
  • welcome is an arbitrary name we are giving to the function.
  • { and } indicates the start and end of the code to be executed, respectively. The code between these symbols will be executed the function is when called.

If we apply the example above to the HTML template, we won't see any changes, as the function hasn't been called yet. But if we add the following line:

welcome()

Then the document will be updated:

🏛️ Hello, student!

Welcome

We can also get a function called when an HTML element is clicked. In order to do so, first we need to get the target HTML element, and then bind the function to the "click" using the addEventListener function.

let element = document.querySelector("p");
element.addEventListener("click", welcome)

From now on, each time we click on first paragraph (🏛️ Hello, student!), the welcome function will be called and then the text Welcome will be appended to the end of the body.
